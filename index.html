Firebase 데이터베이스 연결
부서원 전체가 같은 데이터를 공유하려면 Firebase를 연결하세요.
최초 1회만 설정하면 이후 자동 연결됩니다.
① Firebase 프로젝트 생성
console.firebase.google.com → 프로젝트 추가 → 생성
② Firestore 활성화
Firestore Database → 데이터베이스 만들기 → 테스트 모드 → asia-northeast3(서울)
③ 웹 앱 등록
설정 ⚙ → 내 앱 → </> → 앱 등록 → firebaseConfig 값 입력
