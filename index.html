Firebase 데이터베이스 연결
부서원 전체가 같은 데이터를 공유하려면 Firebase를 연결해야 합니다.
최초 1회만 설정하면 이후엔 자동으로 연결됩니다.
① Firebase 프로젝트 생성
console.firebase.google.com → 프로젝트 추가 → 이름 입력 → 생성 완료
② Firestore 데이터베이스 활성화
좌측 메뉴 Firestore Database → 데이터베이스 만들기 → 테스트 모드 → 리전 asia-northeast3(서울) → 완료
③ 웹 앱 등록 후 설정값 입력
프로젝트 설정 ⚙ → 내 앱 → 웹 아이콘 </> → 앱 등록 → firebaseConfig 값을 아래에 붙여넣기
